Codex marketplace plugin

GoDaddy Domains MCP for Codex

Check domain availability and brainstorm domain ideas from Codex using GoDaddy's public read-only MCP endpoint.

$ codex plugin marketplace add taklegko/codex-godaddy-mcp

After adding the marketplace, install and enable GoDaddy Domains from Easy Codex Plugins in Codex plugin settings.

What it does

  • Checks availability for one or more domains.
  • Suggests domain ideas from keywords or product descriptions.
  • Helps compare naming options.

Read-only

This wrapper does not buy domains, change DNS, inspect account data, or perform GoDaddy account actions.

Example prompts

"Check if crispforge.com is available."

"Find domain ideas for a privacy-first analytics product."

"Find available domains for a privacy-first analytics product."

"Suggest short .com domains for an indie macOS utility."

"Compare crispforge.com, crispforge.io, and getcrispforge.com."
